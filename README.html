Ratakor's Parabola GNU/Linux-libre config

Installation

As if you just installed Parabola OpenRC, DO NOT TRY THIS if it's not a new install and you don't know what you're doing. It will install all my config, scripts and packages except joplin and ungoogled-chromium. Also I follow the .local convention so be sure that PREFIX is setup the way you like.

$ git clone https://github.com/ratakor/config.git
$ cd config
$ make

Softwares

Graphical environment

  • Window manager: dwm
  • Compositor: picom, use xcompmgr if you experience performance issues
  • Bar: dwmblocks
  • Terminal emulator: st
  • Program launcher: dmenu
  • Screen locker: slock

Fonts

  • Serif font: Noto Serif
  • Sans-serif font: Noto Sans
  • Monospace font: Agave Nerd Font Mono
  • Emoji font: Noto Emoji

Miscellaneous

  • Web Browser: Ungoogled Chromium
  • Shell: zsh
  • Editor: Neovim
  • File manager: lf
  • Document viewer: zathura
  • Video player: mpv
  • Image viewer: sxiv
  • Screenshot tool: scrot
  • Notes: Joplin
  • Calendar: when
  • RSS Reader: Newsboat

Software used but not worth mentionning

yt-dlp, tlp, redshift, xbanish, cli-visualizer, neofetch, mesofetch, exa, opendoas, man-db, xdo, htop, oneko, sx, pinta
